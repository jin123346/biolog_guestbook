# 소스코드 보호 빌드 가이드

## 개요
이 빌드 스크립트는 JavaScript와 CSS 파일을 난독화/압축하여 소스코드를 보호합니다.

## 설치

필요한 패키지를 설치합니다:
```bash
npm install
```

## 빌드 실행

```bash
npm run build
```

또는 직접 실행:
```bash
node build.js
```

## 결과

빌드가 완료되면 `dist` 폴더가 생성됩니다:

```
dist/
├── public/
│   ├── index.html          (스크립트 경로가 .min.js, .min.css로 변경됨)
│   ├── script2.min.js      (난독화 및 압축된 JavaScript)
│   ├── style.min.css       (압축된 CSS)
│   └── [이미지 파일들]
├── server.js               (서버 파일 - 그대로 복사)
├── data/
│   └── guestbook.json      (데이터 파일)
└── package.json            (패키지 정보)
```

## 배포

1. `dist` 폴더 전체를 전달하세요
2. 받는 사람은 `dist` 폴더에서 `npm install` 후 `npm start`로 실행하면 됩니다
3. 원본 소스코드는 `dist` 폴더에 포함되지 않습니다

## 주의사항

- **원본 파일 백업**: 빌드 전에 원본 파일을 백업하세요
- **난독화된 코드는 읽기 어렵지만 완전히 숨길 수는 없습니다**
- 브라우저 개발자 도구로 일부 코드는 확인 가능합니다
- 민감한 정보(API 키, 비밀번호 등)는 서버 사이드에서만 처리하세요
