# Node.js 설치 가이드

전자 방명록을 실행하려면 Node.js가 필요합니다.

## 설치 방법

### 1. Node.js 다운로드

**공식 웹사이트**: https://nodejs.org/

- **LTS 버전** (권장) - 안정적인 버전
- **Current 버전** - 최신 기능 포함

### 2. 설치 파일 실행

1. 다운로드한 `.msi` 파일을 더블클릭
2. 설치 마법사가 열리면 **"Next"** 클릭
3. 라이선스 동의 후 **"Next"** 클릭
4. 설치 경로는 기본값 그대로 두고 **"Next"** 클릭
5. **"Add to PATH"** 옵션이 체크되어 있는지 확인 (기본적으로 체크됨)
6. **"Install"** 클릭하여 설치 시작
7. 설치 완료 후 **"Finish"** 클릭

### 3. 설치 확인

**명령 프롬프트** (cmd) 또는 **PowerShell**을 열고 다음 명령어 입력:

```bash
node --version
```

설치가 완료되었다면 버전 번호가 표시됩니다 (예: `v20.10.0`)

```bash
npm --version
```

npm 버전도 확인할 수 있습니다 (예: `10.2.3`)

## 설치 후 해야 할 일

1. **명령 프롬프트를 다시 시작** (설치 후 PATH가 업데이트되므로)
2. `실행하기.bat` 파일을 더블클릭하여 전자 방명록 실행

## 문제 해결

### "node는 내부 또는 외부 명령..." 오류가 나는 경우

1. 명령 프롬프트를 **완전히 종료**하고 다시 열기
2. 그래도 안 되면 컴퓨터를 **재시작**
3. 여전히 안 되면 Node.js를 다시 설치하고 **"Add to PATH"** 옵션 확인

### 설치가 안 되는 경우

- 관리자 권한으로 설치 파일 실행
- 바이러스 백신 프로그램 일시 중지 후 설치
- Windows 업데이트 확인

## 참고

- Node.js는 무료로 제공됩니다
- 설치 후 컴퓨터를 재시작할 필요는 없지만, 명령 프롬프트는 다시 열어야 합니다
- Node.js와 함께 npm (Node Package Manager)도 자동으로 설치됩니다
